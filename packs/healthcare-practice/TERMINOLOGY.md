# Healthcare practice terminology

| Term | Operational meaning |
|---|---|
| Designated system | Approved environment allowed to hold the relevant PHI |
| Minimum necessary | Smallest field set and time window required for a specific administrative purpose |
| Routing | Administrative assignment, not clinical triage or diagnosis |
| Referral loop | Request, scheduling, visit status, records receipt, and human-reviewed follow-up |
| Prior authorization | Payer administrative process; clinical assertions remain clinician-owned |
