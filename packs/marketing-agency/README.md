# Marketing agency pack

This pack turns the reusable operating system into the working environment for **Meridian Creative**, a fictional 12-person agency. It adds agency vocabulary, stricter communication rules, six executable skills, six tool contracts, and realistic operating memory without changing `core/`.

Agents read `COMPANY.md`, `GOVERNANCE.md`, and `TERMINOLOGY.md` when the pack is active. They read a skill only when its trigger matches. They may append session journals and propose new facts, but a human approves durable memory and external sends.

Mock mode uses fixed fixture IDs and dates. It contains no live customer records or credentials.
