# Automation contract

Automations connect typed triggers to versioned workflow steps. Agents may propose drafts; humans validate and publish them according to governance. Every workflow declares trigger schema, owners, deterministic and model steps, tool permissions, approval gates, concurrency, retries, idempotency, outputs, and failure escalation.

Schedules, webhooks, app events, database events, chat requests, and task transitions normalize into a common event envelope before execution.
